# Gara-manager-dev-front-end
## 1. Về kỹ thuật
### a) Cài đặt
- cài đặt nodeJS v21.2.0
- clone git: 
`git clone https://github.com/vkq2808/gara-manager-dev-front-end.git`
- cài đặt các dependencies (node_modules): vào terminal của thư mục gốc, chạy:
`npm i`
- test thử: ` npm start`
